Yet another Geo-IP project

So, I just needed something simple but that works. A simple Geo-IP service that supports JSONP calls

Usage

The application was pretty much designed to be called from javascript, so json is the only response format.

The application will detect if the parameter callback is present and reply using that function, otherwise the json string is returned. Also, if the parameter ip is not present, the application will use the client ip.

jQuery.ajax({
	url: '{{ hostname }}location.json',
	dataType:"jsonp" ,
	success: function(data) {
		// do what your need with your data	
	}});
